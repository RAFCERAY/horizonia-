# 🌐 HORIZONIA

**La plateforme française d'éducation à l'IA pour les 10-17 ans.**

> *« Une génération qui subit l'IA sera gouvernée par elle.
> Une génération qui la maîtrise bâtira l'avenir. »*

La loi sur la majorité numérique, adoptée le 21 juillet 2026, redéfinit la place
des jeunes dans l'espace numérique. HORIZONIA propose la suite : transformer le
temps d'écran en compétence, et faire de l'IA un objet d'apprentissage plutôt
qu'une boîte noire subie.

## Les quatre piliers

**🎓 ARIA — Tuteur IA socratique** — Un agent pédagogique fondé sur des modèles
français (Mistral) qui guide l'élève par le questionnement, sans jamais faire le
travail à sa place. Adaptatif du CM2 à la Terminale, en complément de
l'introduction de l'IA dans les programmes scolaires.

**🎨 Ateliers de création avec l'IA** — Musique, écriture, code, design : les
jeunes passent du statut de consommateurs d'algorithmes à celui de créateurs
outillés.

**🔍 Pédagogie de l'algorithme** — HORIZONIA explique aux élèves comment
fonctionnent les systèmes qu'ils utilisent, au service de l'esprit critique.

**🛡️ Deep-Care — Sécurité dès la conception** — Détection des situations de
harcèlement et de détresse, conçue dans une logique de conformité exigeante
(RGPD, recommandations CNIL pour les mineurs).

## Souveraineté

Modèles français (Mistral) · Hébergement en France · Zéro publicité ·
Zéro exploitation des données · Marque déposée (INPI)

## Statut du projet

Prototype fonctionnel en phase de test · Pilote en établissements scolaires en
préparation pour la rentrée 2027 · Le code source est développé en privé.
